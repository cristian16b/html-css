¿Que significan las siglas HTML y para que se utiliza dicho lenguaje?

HTML signifca HyperText Markup Language lo cual significa “lenguajes de marcas de hipertexto”. Es un lenguaje de etiquetas (tags) que se usa para dar estructura a un documento web que se puede distribuir por internet. Las etiquetas son variadas como input, div, header, etc. Se puede decir que html es como si fueran los huesos del cuerpo humano, luego con css se le da la apariencia exterior. En su version original de los años 90' tenia 22 elementos y se fue ampliando con la nueva version de html5

¿Cuales son las capas de una arquitectura de 3 capas? Fundamente brevemente para que sirve cada una.

Una arquitectura de tres capas son la presentacion de la informacion, logica del negocio y modelo de datos. La presentacion es lo que el usuario ve, en los formatos que sean requeridos (formatos numericos, texto, etc), La logica de negocio indican las reglas de lo que debe hacer el sistema (comportamiento) y el modelo de datos, es como el sistema guarda la informacion de los registros.
